# Быстрый старт

## 🖥️ Минимальные требования

- Любой современный браузер (Chrome, Firefox, Safari, Edge)
- Текстовый редактор для редактирования кода (VS Code, Sublime Text и т.д.)

## 📥 Установка

### Способ 1: Простой (Рекомендуется)

1. Откройте папку `fintech-website` в проводнике
2. Дважды нажмите на `index.html`
3. Сайт откроется в браузере по умолчанию

### Способ 2: VS Code + Live Server

1. Откройте папку проекта в VS Code
2. Установите расширение **Live Server** (если не установлено):
   - Левая панель → Extensions
   - Поиск: "Live Server"
   - Нажмите Install

3. Кликните правой кнопкой на `index.html`
4. Выберите "Open with Live Server"
5. Сайт откроется с автоматической перезагрузкой при изменении кода

### Способ 3: Python (для Windows)

Если установлен Python 3:

```bash
cd path\to\fintech-website
python -m http.server 8000
```

Затем откройте браузер: `http://localhost:8000`

## 🎮 Как использовать

### Главная страница
- Нажмите логотип **FinFlow** в левом верхнем углу
- Или клавишу на главную в меню

### Магазин
1. Перейдите в раздел **Магазин**
2. Выбирайте категории с помощью кнопок
3. Используйте ползунок для фильтра по цене
4. Нажимайте **Купить** для добавления товара в корзину

### Платежи
1. Перейдите в **Платежи**
2. Выберите тип услуги из выпадающего списка
3. Введите номер счета и сумму
4. Нажмите **Оплатить**

### Переводы
1. Перейдите в **Переводы**
2. Введите номер телефона получателя
3. Укажите имя и сумму
4. Комиссия считается автоматически (1%)
5. Нажмите **Отправить**

### Профиль
1. Перейдите в **Профиль**
2. Зарегистрируйтесь или войдите
3. Тестовый аккаунт: любые данные (сохраняются в памяти браузера)
4. После входа - просмотр профиля и выход

## 📂 Структура файлов

```
fintech-website/
├── index.html              # Главный HTML файл
├── css/
│   └── style.css          # Все CSS стили
├── js/
│   └── script.js          # JavaScript логика
└── README.md              # Документация
```

### Что находится в каждом файле?

**index.html**
- Header с логотипом и навигацией
- Footer со ссылками
- Контейнер для динамического контента
- Подключение CSS и JS

**css/style.css**
- CSS переменные для цветов
- Стили компонентов (кнопки, карточки, формы)
- Медиа-запросы для адаптивности
- Анимации и переходы

**js/script.js**
- Объект `app` с состоянием приложения
- Функции навигации между страницами
- Рендеринг HTML для каждой страницы
- Обработчики событий форм
- Фильтрация товаров

## 🎨 Как редактировать?

### Изменить цвета
Откройте `css/style.css` и найдите секцию `VARIABLES`:

```css
:root {
    --primary-color: #e63946;    /* Красный */
    --secondary-color: #457b9d;  /* Синий */
    --success-color: #2ecc71;    /* Зелёный */
    /* и т.д. */
}
```

### Добавить новый товар
Откройте `js/script.js` и найдите массив `app.products`:

```javascript
{
    id: 9,
    name: 'Новый товар',
    category: 'electronics',
    price: 99999,
    image: '🆕',
    rating: 4.5,
    reviews: 100
}
```

### Изменить текст на странице
Найдите нужную функцию `render*()` в `script.js` и отредактируйте HTML строку.

## 🆘 Возможные проблемы

### Проблема: Сайт не открывается
**Решение:** Убедитесь, что вы открываете `index.html`, а не другой файл

### Проблема: Стили не применяются
**Решение:** 
-清除 кэш браузера (Ctrl+Shift+Del)
- Обновите страницу (Ctrl+F5)

### Проблема: Меню не работает на мобильном
**Решение:** Это учебная версия - бургер-меню не реализовано

### Проблема: Данные теряются при обновлении
**Решение:** Это нормально - данные хранятся только в памяти браузера
