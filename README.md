# ILYAS BANK

Современное демо-SPA банковского приложения в красно-белой финтех-стилистике.

## Что есть

- Главная с быстрыми банковскими сервисами
- Вход и регистрация в демо-режиме
- Личный кабинет с балансом и картой
- Переводы
- Платежи
- Карты с оформлением, лимитами и блокировкой
- Кредиты с калькулятором, графиком и погашением
- Депозиты с расчетом дохода и пополнением
- QR-оплата
- История операций с фильтрами и экспортом
- Профиль и смена пароля
- Поддержка с FAQ и демо-чатом

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 8000
```

Затем откройте:

```text
http://127.0.0.1:8000/
```

## Демо-вход

Можно войти любым email и паролем:

```text
Test1234!
```

Данные сохраняются в `localStorage` текущего браузера.
