# 🏦 ILYAS BANK - Production-Ready Fintech

## 🚀 Быстрый старт

```bash

# Зарегистрируйтесь: test@mail.com / Test1234!

```

### С Backend (полная версия)
```bash
cd backend
npm install
npm start

# Server запустится на http://localhost:3000
```

## 📄 Страницы

### 1. **Главная (Home)**
- Красивый hero баннер с кнопками действия
- Три основных сервиса (карточки)
- Преимущества платформы
- CTA секция для регистрации

### 2. **Магазин (Shop)**
- Каталог товаров (8 товаров)
- Картинки товаров (эмодзи как плейсхолдеры)
- Фильтрация по категориям:
  - Все товары
  - Электроника
  - Аксессуары
  - Одежда
- Фильтр по цене (ползунок)
- Рейтинги и отзывы
- Кнопка "Купить"

### 3. **Платежи (Payments)**
- Форма оплаты коммунальных услуг
- Выбор типа услуги (электричество, вода, газ, интернет, телефон, ТВ)
- Ввод номера счета и суммы
- Карточки популярных услуг

### 4. **Переводы (Transfers)**
- Форма для отправки денег
- Номер телефона получателя
- Имя получателя
- Расчет комиссии (1% от суммы)
- История последних переводов

### 5. **Профиль (Profile)**
- **До входа:**
  - Форма входа (email + пароль)
  - Форма регистрации (имя, email, телефон, пароль)
- **После входа:**
  - Информация пользователя
  - Баланс счета
  - Кнопки редактирования профиля
  - Кнопка выхода

## ⚡ Функциональность JavaScript

### Компоненты
- **SPA (Single Page Application)** - без перезагрузки страницы
- **Динамическая навигация** - переключение между страницами
- **Управление состоянием (State Management)** - сохранение данных в `app` объекте
- **Формы** - обработка отправки формы, валидация
- **Фильтрация** - фильтрация товаров по категории и цене
- **Уведомления** - всплывающие сообщения об успехе/ошибке

### Основные функции
```javascript
// Навигация
navigateTo(page)          // Переключение между страницами
loadPage(page)            // Загрузка содержимого страницы

// Продукты и фильтры
filterProducts(category)  // Фильтрация товаров
addToCart(productId)      // Добавление в корзину

// Формы
initPaymentForm()         // Инициализация формы платежей
initTransferForm()        // Инициализация формы переводов
initProfileForm()         // Инициализация форм профиля

// Утилиты
formatPrice(price)        // Форматирование цены
showNotification(msg)     // Показ уведомлений
```

## 📱 Адаптивность

Сайт полностью адаптивен:
- **Десктоп (1200px+):** Полная версия с боковой навигацией
- **Планшет (768px - 1200px):** Оптимизированная сетка
- **Мобильный (<768px):** Компактная версия с бургер-меню

- **HTML5** - семантическая структура

- **Vanilla JavaScript** - чистый JS без фреймворков
